# Skill

A packaged capability the agent can load and use. Progressive loading: agent sees name + description first, loads full instructions on activation.

Minimal definition (`SKILL.md` frontmatter):
- `name` — skill identifier
- `description` — what it does, when to use it (for LLM to decide)

Reference: https://agentskills.io/specification
